# Technology Stack\n\n## Language\n- **Python:** The primary programming language used for both core logic and the web application backend.\n\n## Backend Framework\n- **Flask:** A lightweight and flexible WSGI web application framework for Python, ideal for building simple yet powerful control interfaces.\n\n## Core Libraries & APIs\n- **picamera:** Provides a pure Python interface to the Raspberry Pi camera module, allowing direct control over various camera settings.\n- **numpy:** Used for efficient numerical computations and array manipulations, likely involved in image processing or plate solving.\n- **pillow (PIL Fork):** Provides image processing capabilities, such as resizing, cropping, and format conversion.\n- **smbus2:** A library for I2C communication, potentially used for interfacing with other sensors or hardware components.\n\n## Hardware Platform\n- **Raspberry Pi:** The target hardware platform for running the findr application and controlling the telescope camera.\n
